|- Blog

## ディレクトリ構成
|--- src
    |--- components //　コンポーネントを格納してください
    |--- pages // ページファイルを格納してください。
    |--- styles // グローバルスタイルを格納してください。
    |--- utils // ユーティリティファイルを格納してください。
    |--- assets // アセットファイルを格納してください。
    |--- layouts // レイアウトファイルを格納してください。
|--- public
|--- docs

## 開発ルール
- 必ずTypeScriptを使用してください。
- 必ずTailwindCSSを使用してください。 
- 必ずGitを使用してください。
- 必ずディレクトリ構成を遵守してください。
- レスポンシブ対応をお願いします。
    - 対応する最小限のブラウザは以下の通りです。
        - Chrome
        - Safari
        - Firefox
        - Edge
    - レスポンシブ対応は以下の通りです。
        - PC
        - Tablet
        - Mobile
- 開発手順の順番に対応をお願いします。
- 必ずドキュメントを残してください。(例:top.md, about.md, works.md, contact.md)
- 1つのタスクを完了するまで、コミットをしないでください。
