# Works Page Documentation

## Overview
The Works page showcases the portfolio of projects.

## Design
- **Grid Layout**: Responsive grid displaying project cards.
- **Project Card**: Contains project thumbnail, title, description, and link.

## Components
- `Card`: Reusable component for displaying project info.

## Implementation Details
- File: `src/pages/works.astro`
- Layout: `src/layouts/Layout.astro`
- Data: Static array of projects for now.
