# TOP Page Documentation

## Overview
The TOP page serves as the entry point to the portfolio. It provides a brief introduction and navigation to other sections (About, Works, Contact).

## Design
- **Hero Section**: A large, welcoming area with the user's name and a tagline.
- **Navigation**: Clear links to other pages.
- **Aesthetic**: Minimalist, premium feel using TailwindCSS.

## Components
- `Header`: Contains the navigation menu.
- `Footer`: Contains copyright and social links.
- `Hero`: The main introductory section.

- File: `src/pages/index.astro`
- Layout: `src/layouts/Layout.astro`
